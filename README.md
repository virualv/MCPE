# Minecraft Bedrock Edition (原Pocket Edition) Backup Package
*Powered by Nukkit*

## 安装
### 依赖
* Java运行环境（JRE）
### 步骤
1. 拉取仓库：
	git clone https://github.com/virualv/MCPE.git
2. 进入目录：
	cd MCPE
3. 赋予启动脚本执行权限，并启动
	chmod a+x start.sh
	./start.sh
